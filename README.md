# Quote Generator

A simple React quote generator that fetches inspirational quotes, lets users translate them, and copy them with one click.

## Features

- Fetches random inspirational quotes from an external API
- Shows the quote author
- Translates quotes into English, French, or Arabic
- Copies the displayed quote to the clipboard
- Built with React and Vite

## Tech Stack

- React
- Vite
- Font Awesome

## Project Idea

This app is a small practice project focused on working with APIs, React state, user interactions, and clean UI structure.
